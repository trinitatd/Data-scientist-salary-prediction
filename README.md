# 💼 Data Scientist Salary Predictor

A machine learning web application that predicts data science salaries based on role, experience, and company attributes.

---

## 🌐 Live Demo

👉 https://data-scientist-salary-predictor.streamlit.app/

---

## 📌 Overview

This project is an end-to-end machine learning application built using **Streamlit**.
It allows users to input job-related details and receive an estimated salary prediction in real-time.

The model is trained on real-world data science salary datasets and captures how different factors influence compensation.

---

## 🚀 Features

* 🔮 Predict salaries instantly based on user input
* 🧠 Machine Learning model (Linear Regression)
* 📊 Clean and interactive UI using Streamlit
* 🌍 Deployed and accessible online
* ⚡ Fast and user-friendly experience

---

## 🧠 Model Details

* **Algorithms used:** Linear Regression, Random Forest
* **Final model selected:** Linear Regression
* **Reason:** Higher performance with R² score ≈ **0.83**
* **Libraries used:** scikit-learn, pandas, joblib

---

## 📥 Input Parameters

The prediction is based on the following features:

* 👨‍💻 Experience Level (EN, MI, SE, EX)
* 🏢 Employment Type (FT, PT, CT, FL)
* 💼 Job Title
* 🌍 Employee Residence
* 🏠 Remote Ratio (0–100)
* 📍 Company Location
* 🏬 Company Size (S, M, L)

---


## 🖥️ Usage

1. Open the app in your browser
2. Enter the required job details
3. Click on **Predict Salary**
4. View the estimated salary instantly
